# PMMSherpa Demo Plan

> **3 Punchy Video Demos for Homepage & Email Distribution**

---

## Overview

These demos are designed to be:
- **Under 60 seconds** each
- **Embeddable** on the PMM Sherpa homepage
- **Shareable** via email campaigns
- **Self-explanatory** with captions (no voiceover required)

---

## Demo 1: "The Positioning Emergency" (45-50 seconds)

### Hook
*"Your VP just asked for a positioning statement. You have 60 seconds."*

### Script/Flow

| Time | Action | On Screen |
|------|--------|-----------|
| 0-5s | Type or speak prompt | "Create a positioning statement for an AI-powered customer success platform that predicts churn before it happens" |
| 5-15s | System processes | Status: "Searching PMM knowledge base..." → Citations loading |
| 15-40s | Response streams | COMPLETE positioning statement (no "[insert here]" placeholders) |
| 40-45s | Copy action | Click "Copy for Google Docs" → Paste into Google Doc with perfect formatting |
| 45-50s | End card | "Expert knowledge. Production-ready output. Your second brain for PMM." |

### Why It Works
- Shows the core value prop in under a minute
- Demonstrates zero-placeholder output (a huge differentiator vs ChatGPT)
- Ends with a satisfying "job done" moment
- Relatable scenario every PMM has experienced

---

## Demo 2: "The Competitive Intel Crunch" (50-60 seconds)

### Hook
*"Your competitor just launched. Leadership wants a response by EOD."*

### Script/Flow

| Time | Action | On Screen |
|------|--------|-----------|
| 0-5s | Paste competitor URL | Type company name or paste their website URL |
| 5-10s | Auto-detection | Web search auto-enables (callout: "it knows you need live data") |
| 10-15s | Select research mode | Click Perplexity dropdown → Select "Deep Research" |
| 15-35s | Parallel processing | Show dual status: RAG + Perplexity running simultaneously |
| 35-50s | Response streams | Expert frameworks (with citations) + Live competitive data (with web sources) |
| 50-60s | Expand research | Click to show Web Research panel with all sources |
| End | End card | "Expert strategy + live intel. Synthesized in seconds." |

### Why It Works
- Shows the unique RAG + Perplexity parallel architecture
- Addresses a real, time-sensitive PMM pain point
- Demonstrates intelligent auto-detection of research needs
- Combines timeless frameworks with real-time data

---

## Demo 3: "From Chaos to Battlecard" (55-65 seconds)

### Hook
*"You just got 47 pages of competitor docs dumped on your desk. Sales needs a battlecard tomorrow."*

### Script/Flow

| Time | Action | On Screen |
|------|--------|-----------|
| 0-8s | Upload file | Click paperclip → Select "competitor_whitepaper.pdf" → Show attachment |
| 8-15s | Type prompt | "Create a competitive battlecard comparing us to this competitor. Focus on enterprise security and pricing" |
| 15-45s | Response streams | Complete battlecard with: Positioning comparison, Key differentiators, Objection handling, Proof points |
| 45-55s | Show citations | Expand panel: "Based on April Dunford's competitive positioning framework..." + "From your uploaded document..." |
| 55-60s | Copy output | Click "Copy as Markdown" → Show it's ready for Notion/Confluence |
| End | End card | "47 pages → 1 actionable battlecard. In under a minute." |

### Why It Works
- Shows multimodal capability (file upload)
- Demonstrates practical deliverable generation
- Massive time savings are visually obvious
- Sales enablement is a universal PMM responsibility

## Production Recommendations

### Technical Specs

| Aspect | Recommendation |
|--------|----------------|
| **Recording tool** | Loom, Screen Studio, or Cleanshot X |
| **Resolution** | 1920x1080 minimum, 4K preferred |
| **Frame rate** | 60fps for smooth scrolling |
| **Format** | MP4 (H.264) for web, WebM as fallback |

### Editing Guidelines

| Element | Guidance |
|---------|----------|
| **Audio** | Subtle lo-fi background music, no voiceover |
| **Pacing** | 1.5x speed for any "waiting" moments |
| **Zoom** | Gentle zoom on key UI elements (citations, copy menu) |
| **Captions** | Large, high-contrast, bottom-center |
| **Cursor** | Highlighted cursor for visibility |

### Branding

| Element | Style |
|---------|-------|
| **Intro** | 2-second PMMSherpa logo fade-in |
| **End card** | Gradient background + CTA button animation |
| **Colors** | Use brand gradients (teal-cyan, indigo-purple) |
